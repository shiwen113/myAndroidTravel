能写文件拷贝
java.io
1、File:文件或目录，与平台无关
    new File("相对或绝对文件名")  
    一些常见方法

2、理解流(stream)：水流，流动的是数据
数据源：文件、网络
        标准输入：System.in:键盘
            输出：System.out,显示屏

流的分类：
方向:相对于程序，输入流，程序从流中读数据
                 输出流，程序中写数据
数据单位：字节（1byte=8bit）、字符
功能：节点流、处理流

类：java.io

3、抽象类（4） 重点关注：读写方法
              字节               字符
  输入      InputStream         Reader     3个read     close
  输出      OutputStream        Writer     4个writer   close  flush

4、节点流：文件，创建对象的方法
        
            字节                    字符
  输入      FileInputStream         FileReader
  输出      FileOutputStream        FileWriter

    输入输出:new FileXxx("File对象 或文件路径")
    输出：new FileOutputStream|FileWriter("File对象 或文件路径",false|true)
        false:覆盖  true:append
    InputStream in = new FileInputStream("....");


5、处理流（包装类）：特征：构造方法以相应的流对象为参数
  23设计模式


1)带缓存的处理流：Buffered
            BufferedInputStream    BufferedReader
            BufferedOutputStram    BufferdeWriter
  1)如何创建这个对象：类似，传一个对应的（）流对象
  2)增加方法：
     BufferedReader:  readLine：读一行文本
     BufferedWriter:  newLine


new BufferedReader(
	new InputStreamReader(
		new FileInputStream(
			new File(""))))

2)其它处理流：PrintStream  PrintWriter, 包装的是 OutputStream
   提供了对应的print方法  print
                          println
                          printf
System.in   类型  InputStream  标准输入
System.out  类型  PrintStream  标准输出

3)InputSreamReader    字节输入流 ==》字符输入流
  OutputStreamWriter  字节输出流 ==》字符输出流

6、RandomAccessFile特点：
1）、不是体制中，访问指针，移动指针可以到文件的一个特殊的地方读写
2）、提供了readXxx和writerXxx方法
   Xxx：基本数据类型

7、读写文件的基本流程：
   1）创建相应的对象
      用Buffered对节点流做包装
   2）用read方法或writer方法读写数据
   3）关闭资源
  关注：异常的处理
     FileNotFoundException、IOException(编译期异常)

常用接口：Cloneable  Comparable  Serializable(标志接口)

8、对象的序列化与反序列化
1）基本概念：
   序列化：Object ==> 二进制流 ==> 存在文件中或送到网络的另一端
  反序列化：从文件中读出二进制流 ==》一个Java对象
  对象序列化必须实现Serializable接口
2）用到两个类：
ObjectOutputStream 处理流，writeObject，序列化
ObjectInputStream  处理流, readObject,反序列化

3）理解transient关键字的含义，相应的字段不序列化

4）（**）自定义序列化流程
面试题:创建对象的几种方法： new clone Factory method 反序列化  反射
